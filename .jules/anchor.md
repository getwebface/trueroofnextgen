## 2024-05-24 - Injecting HTML Trust Blocks into Astro Markdown **Learning:** Astro's markdown parser handles raw HTML `<div>` blocks (like guarantee boxes with inline styles and SVG icons) perfectly without breaking the surrounding layout or causing hydration errors, provided they are separated by blank lines from the markdown text. **Action:** When injecting E-E-A-T signals like guarantee boxes into existing location or service pages, confidently use raw HTML blocks with inline CSS right inside the `.md` files instead of creating complex `.astro` or React component wrappers, as it's cleaner and faster.
