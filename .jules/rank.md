## 2024-05-24 - [Astro React 19 Edge SSR Rendering Quirks] **Learning:** Attempting to verify server-side rendered SEO meta tags via `pnpm run dev` with Astro + Cloudflare Adapter + React 19 resulted in `require is not defined` errors originating from `react-dom/server.edge.js`. This is a known issue with the local Vite dev server failing to replicate the Edge environment perfectly. **Action:** When verifying SEO tags or SSR functionality on the Cloudflare pages adapter, always build the project (`pnpm run build`) and use `npx wrangler pages dev dist` rather than relying on the Astro local dev server (`npm run dev`).
