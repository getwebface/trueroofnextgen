## 2025-03-01 - [Astro Markdown HTML injection] **Learning:** When wrapping markdown in HTML tags (like `<div class="glass-card">`) in Astro, ensure there are blank lines before and after the opening and closing HTML tags to guarantee the inner markdown parses correctly. **Action:** Always use blank lines around injected HTML blocks in Astro content collections.
## 2025-03-01 - [Local Preview Port] **Learning:** The default preview command doesn't work locally for the Cloudflare adapter. Using `wrangler pages dev dist` exposes the site on port 8788. **Action:** Use `wrangler pages dev dist` instead of `astro preview` and verify against port 8788.
