## 2025-03-01 - [Astro Markdown HTML injection] **Learning:** When wrapping markdown in HTML tags (like `<div class="glass-card">`) in Astro, ensure there are blank lines before and after the opening and closing HTML tags to guarantee the inner markdown parses correctly. **Action:** Always use blank lines around injected HTML blocks in Astro content collections.
## 2025-03-01 - [Local Preview Port] **Learning:** The default preview command doesn't work locally for the Cloudflare adapter. Using `wrangler pages dev dist` exposes the site on port 8788. **Action:** Use `wrangler pages dev dist` instead of `astro preview` and verify against port 8788.
## 2025-03-01 - [Strict Tailwind Enforcement] **Learning:** Structural changes using inline CSS get rejected because it violates the strict "Tailwind mobile-first classes" constraint. **Action:** Always use Tailwind utility classes (e.g., `p-6`) instead of inline styles for layout enforcement.
## 2024-05-24 - Wrapped Markdown Content in strict max-w-prose **Learning:** Unconstrained markdown rendering in Astro can lead to lines that span the full width of the grid column, making reading difficult and disrupting the F-pattern. **Action:** Always wrap Astro `<Content />` blocks in `max-w-prose` combined with a responsive Grid layout to establish an explicit and comfortable F-pattern reading constraint.
